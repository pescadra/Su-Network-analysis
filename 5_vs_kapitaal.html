Cat. 3b: EBITDA-range met equity bridge¶

Methodologie (professionele waarderingsaanpak):

  1. LTM EBITDA: Laatste boekjaar vóór exit-datum uit nbb_prd.gold.nbb_jaarrekening (≤2 jaar gap)
  2. 3 scenario's: 🔴 Bear (3×) / 🟡 Base (5×) / 🟢 Bull (8×)
  3. Equity bridge: Equity = EV − kort vreemd vermogen + vlottende activa

Jaarkwaliteit (N=65 matches, 49 betrouwbaar):

Gap exit–NBB N Beoordeling
0 jaar (zelfde jaar) 28 ✅ Betrouwbaar
1 jaar (LTM) 13 ✅ Betrouwbaar
2 jaar 8 ⚠️ Acceptabel met voetnoot
3+ jaar 16 ❌ Uitgesloten (bedrijf kan totaal veranderd zijn)

Resultaat (49 betrouwbare exits):

Scenario EBITDA-multiple Totale equity
🔴 Bear (pessimistisch) 3× €467M
🟡 Base 5× €711M
🟢 Bull (optimistisch) 8× €1.078M

Top exits: DESOTEC (€120–278M), Cmosis (€57–134M), Itineris (€64–128M), Accent Jobs (€20–81M)
⚠️ Caveats: (1) EBITDA niet genormaliseerd. (2) Kort vreemd vermogen als schuldproxy. (3) Vlottende activa als cash-proxy.

Mogelijke verklaringen:

  • Het Vlaamse startup-ecosysteem is kleiner en minder zichtbaar voor internationale PE-fondsen.

4. Beleidsaanbevelingen¶

  1. VS-hefboom aanpakken: ratio 2,7× toont dat VS-kopers disproportioneel profiteren van Vlaamse startups. Retentie-instrumenten overwegen (bv. belastingvoordelen bij langer aanhouden, golden shares bij publiek gefinancierde startups).

  2. Europees patroon benutten: Schengen+GB = #1 buitenlandse koper in aantallen. Europese integratie is een kans — Vlaamse startups schalen vaak via Europese overnames. Beleid kan dit kanaliseren.

  3. Waarde-paradox adresseren: België doet het meeste deals maar vangt slechts 10% van de waarde. Belgische kopers nemen kleinere startups over. Corporate venturing / scale-up programma’s kunnen Belgische bedrijven helpen om grotere deals te doen.

  4. Structureel dashboard: deze analyse is een nulmeting. Kwartaalupdates met Dealroom + Sequenza + NBB data zijn haalbaar binnen het VLAIO-dataplatform.
